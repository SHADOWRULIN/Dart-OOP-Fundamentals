# 🎯 Dart OOP Fundamentals: Midterm Project

This repository contains the implementation of core Object-Oriented Programming (OOP) concepts in Dart. I developed this project as a midterm assessment for the **Mobile App Development - 1** course at **Bano Qabil 2.0**.

## 📝 Project Overview
The project consists of two primary challenges designed by **Instructor Muhammad Salman Bediya** to test proficiency in class structures, attribute management, and logic implementation in Dart:

### 1. BankAccount Management System
* **Attributes**: Account Number (int), Balance (double), Account Type (string), and Interest Rate (double).
* **Functionality**:
    * **Deposit**: Adds a specified amount to the balance.
    * **Withdraw**: Subtracts an amount from the balance, with validation to prevent withdrawing more than the current balance.
    * **Add Interest**: Calculates and adds interest to the balance based on the assigned rate.
    * **Display**: Prints a full summary of the account details.

### 2. Student Information System
* **Attributes**: Name (string), ID (string), and a dynamic list of courses (strings).
* **Functionality**:
    * **Add Course**: Dynamically adds a new course to the student's record.
    * **Drop Course**: Removes an existing course from the student's list.
    * **Display Courses**: Prints a complete list of the student's current academic profile.

## 🛠️ Technical Skills Demonstrated
* **Encapsulation**: Using classes to bundle data and methods together.
* **Logic Validation**: Implementing conditional checks to ensure data integrity.
* **Collection Management**: Manipulating dynamic lists within an object-oriented context.
* **Object Instantiation**: Creating and managing multiple unique instances of the same class.
